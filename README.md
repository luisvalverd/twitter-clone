# Twitter Clone 

this is a aplication is a clone of twitter using VueJs in frontend and server using Express

## start servers


### install dependencies of server

``` bash  
  cd ./server
  npm install
```
### install dependencies of client

``` bash
  cd ./client
  npm install
```

### install dependencies of chat server
``` bash
  cd ./chat-server
  npm install
```

## run servers

### run server
``` bash
  cd ./server
  npm run dev:server
```
### run chat-server
in terminal execute:
``` bash
  cd ./chat-server
  npm run dev:serve
```

### run client
in terminal execute:
``` bash
  cd ./client
  npm run serve
```
